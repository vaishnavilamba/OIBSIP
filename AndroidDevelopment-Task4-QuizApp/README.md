# 🧠 Quiz App

## 📖 Description

A simple Android Quiz Application developed using Kotlin and XML. The app allows users to answer multiple-choice questions, calculates the score, and displays the final result.

---

## ✨ Features

- Multiple Choice Questions
- Score Calculation
- Next Question Navigation
- Result Screen
- User-Friendly Interface

---

## 🛠 Tech Stack

- Kotlin
- XML
- Android Studio

## 🚀 How to Run

1. Clone this repository.
2. Open the project in Android Studio.
3. Sync Gradle.
4. Run the application on an emulator or Android device.
